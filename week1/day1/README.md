## 字典和类的区别
| 维度 | `dict`（字典） | `class`（类） |
|------|----------------|---------------|
| 本质 | 内置数据容器（像一个带标签的盒子） | 自定义数据类型（像一个“模具”或“蓝图”） |
| 结构 | 动态、灵活，随时可增删键 | 固定结构（由 `__init__` 定义），更稳定 |
| 行为 | 只能存数据，不能自带方法 | 可以同时包含数据 + 方法（如 `.to_json()`） |
| 安全性 | 容易拼错键名（如 `"descripton"`） | 属性名固定，IDE 能提示+检查 |
| 可读性 | `task["description"]` | `task.description`（更自然） |
| 扩展性 | 难以复用逻辑 | 可继承、可重用、可测试 |
